# linux-setup

Automates Arch Linux setup after a fresh install.

## Requirements

- Arch Linux
- sudo

## Usage

```bash
sudo ./setup.sh
```

## What it does

- Installs packages (skips already installed)
- Configures git (name, email)

## Packages

| Package | Description |
|---------|-------------|
| git | Version control |
| curl | HTTP requests |
| vim | Text editor |
| htop | Process monitor |
